Introduction

To begin with, thank you for trusting us by purchasing this tool which allows you to create and manage Telegram bots without writing a single line of code.

This documentation helps you install Telegram Bot Builder and gives you the information you need to create, modify, delete a bot. Please we ask you to carefully follow the instructions in this documentation in order to get started with this tool.

Requirements

You will need the following component to use Telegram Bot Builder.

  1. A web server running under the PHP engine
  2. An SSL certificate associated with your server (your server must work with the https protocol)
  3. A MySql database server

How to Install

Upload files

To upload the files to your server, first unzip the source file that was provided to you, then place the files in Telegram-Bot-Builder-Source on your server in the folder you prefer via an FTP server (Filezilla by example).

Change base URL

After uploading the files to your server, you must modify the base URL, to do this modify the content of the application/config/config.php file on line 26 replace the value of $config['base_url'] by the url where you uploaded the files, the url must start with https://.

Example
$config['base_url'] = 'https://yourdomain.extension/folder';

Importing Database

To import the database, you must first import the database.sql script file into your mysql database.

Then contiue by by putting the information of your database in application/config/database.php on line 76 modify the information in the following fields of the array $db['default'] :

  1. hostname : the name of the host of your database server.
  2. username : the username of your database
  3. password : the password of your database
  4. database : the name of your database that you created.

The result in the file might look like:

							$db['default'] = array(
							'dsn'	=> '',
							'hostname' => 'your.mysql.host',
							'username' => 'mysql_username',
							'password' => 'your_password',
								'database' => 'ypur_database_name',
							...

Login

To connect, simply go to the q = link where you have installed the tool and enter the following coordinates:

  • Email: admin@admin.com
  • Password: password
Note that you can modify its coordinates whenever you want in the Manage Users section.

Create bots #back to top

You can create a bot from the home page by clicking on the Create bot button.

Step 1: Getting Started - #back to top

The first step gives you a general overview of what a Telegram bot is and gives you the information you need to start creating a bot with Telegram Bot Builder.

You can add a bot to Telegram bot builder in two ways:

  • By creating a new Telegram bot
  • Using an already existing bot

In both cases start by going to the discussion with BotFather by clicking on BotFather

BotFather as its name suggests, it is the father of all bots, it is a bot that allows you to create bots. Once you are in the conversation with BotFather, follow the instructions below depending on whether you want to add a new bot or modify an existing bot.

Create new bot



To create a new bot from the botfather, write '/' in the edit box of the conversation with Botfather, this will open a list of existing commands, select the command '/newbot'

Follow the instructions that will allow you to create a bot, at the end you will be given your Token, keep this Token well because it will be useful in the next step.

Add bot from an existing one



To create bot from existing one, write '/' in the edit box of the conversation with Botfather, this will open a list of existing commands, select the command '/mybots'

BotFather will then send you the list of bots you own, click on the bot you want to add, in the next menu click on "API Token" Bot Father will send you your bot's Token, which will be important in the next step.

Once it's done you can move on to the next step.

Step 2: Basic informations - #back to top

In this step you will fill in the information you got from BotFather:

  1. YOUR BOT USERNAME (The same username you provides to Telegram)
  2. YOUR BOT API TOKEN (Your bot Token given by Telegram)
  3. YOUR BOT NAME (This information is specific to Telegram Bot builder, you can put the value you want)

Step 3: Set bot description - #back to top

If you haven't already done so, it is important to define the description of your bot, this description will be the first message displayed when the user opens your bot for the first time.

If you haven't already, go into the conversation with our dear friend Bot Father next:

  1. Click on the command /mybots
  2. Select your bot
  3. Click on Edit bot
  4. Choose Edit Description
  5. Send description text to Bot Father
  6. It's done your description is changed!

Step 4: Set bot "About" text - #back to top

If you haven't already done so, it is important to define the about text of your bot, this text will be displayed in the user profil of your bot.

If you haven't already, go into the conversation with our dear friend Bot Father next:

  1. Click on the command /mybots
  2. Select your bot
  3. Click on Edit bot
  4. Choose Edit About
  5. Send about text to Bot Father
  6. It's done your about text is changed!

Step 5: Set bot picture - #back to top

If you haven't already done so, it is important to set a profile picture for your bot.

If you haven't already, go into the conversation with our dear friend Bot Father next:

  1. Click on the command /mybots
  2. Select your bot
  3. Click on Edit bot
  4. Choose Edit Botpic
  5. Send a picture to Bot Father
  6. It's done your picture is set!

Step 6: Set start command message - #back to top

In this step you will have to specify the message that the user will see when he uses your bot for the first time by pressing the START button. He will also see this message every time he presses the /start command.

This first message will be accompanied by a menu which will be defined in the next step.

You can uses emoji in your message

Step 7: Add all menus - #back to top

Your bot will constitute different menus which will allow your customers to navigate through the different services that you offer.

The menu represents in a way the keyboard, this one will be placed just below the text zone of message. Define all menus of your bot and the different options it can contain. Actions on menu options will be defined in the next steps.

In this step you are called to create all the possible menus of your bot and the message for each menu that will be displayed when the menu is generated. It is therefore up to you to already design all the menus you want to create and their options.

Example: You can consider creating two menus:

  1. A main menu which will have two options:
    1. An option Our Services which when clicked will display the second menu
    2. An "About Us" option which when clicked will display a message explaining who you are.
  2. A second menu where each option will represent each of your services and when clicked on it will display a message that explains your services and a link for more details.

Note that a menu option can leads to another menu, this will be defined in the next step.

Here is the information to provide to add a menu:

Menu Title

Give a title to your menu, which will identify it for future modifications (Ex: Main menu).

Max buttons per line

The maximum number of buttons per line represents the number of buttons that will be on each line of your keyboard if, for example, you put velor 2 and you have 3 buttons; the first line will have 2 buttons while the last one will have only 1.

Menu Options

Define the options of your menu which represent the buttons which will be displayed on the menus these will have an action which will send a message or generate another menu on click. These actions will be defined in the next steps

To add a menu option, click the Add new Option button and in the modal that appears, enter the following information:

Option title
Message text: The message that will be sent when you click on the menu option
Message Picture (Optional)
Message buttons (Optional): It is possible to add buttons to your message; these will appear below your post and each button will lead to a specific URL.
Max buttons per line: In case you add buttons to your message, this option represents the maximum number of buttons that will be on a line.
Buttons: the buttons of your message which are each composed of a title and a url address. You can add as many buttons as you want.

Once you have added all the information on your menu click on Add Menu to add it. You can add others; after adding all the necessary menus for your bot, click Next to continue.

Step 8: Add action to menu options - #back to top

In this step, you will define the actions on the menu options, i.e. if you have for example a menu 1 which has a menu option 2 which when you click on it displays menu 2, it is in this step that you will define this, if an option does not lead to any other menu, leave it as it is.

The actions will be defined on the options of all the menus each time you click on Next you will go to the options of another menu until all the menus are browsed.

To assign an action to an option, click on the Add action button located in the area of the menu option or want to add the action, then in the modal that is displayed, click on the menu that will be displayed when 'we will click on this option.

Step 9: Set main menu - #back to top

Select the main menu of your bot, the main menu is the entry point of your bot which will be displayed when the user clicks on the /start command

Step 10: Finalization - #back to top

Congratulations, your bot has been created successfully, You can get the link to your bot in the last post.

Update bots #back to top

To modify a specific bot, go to the Chat Bots section then click on the Edit icon of the bot you want to modify.

Finally, you just have to follow the same steps as when creating a bot.

Delete bot #back to top

To remove a specific bot, go to the Chat Bots section then click on the Trash icon of the bot you want to delete.

Finally in the modal that appears, confirm that you really want to delete the bot.

Once you delete the bot, you cannot retrieve any information from it, so be sure before performing this action.

Manage users #back to top

This section concerns the management of Telegram Bot Builder users, there are two types:

  1. The admins who have all the rights and can therefore perform any action.
  2. Simple users who only have the right to send messages to your customers through the bot.

Add users - #back to top

To add a user, go to the Manage Users section then click on the Add User button.

In the modal that appears, enter the information of the new user you want to add:

  1. Username: The username which must be unique.
  2. Email: The email adress which must be unique.
  3. Password: at least 8 characters
  4. First Name
  5. Last Name
  6. User group: Member or Admin

Update user - #back to top

To update the information of a particular user, click on the Edit icon of the user you want to modify.

In the modal that appears, modify the user's information.

If you want to change the user's password, in addition to entering the new password, check the "Edit password" box.

Delete user #back to top

To remove a specific user, go to the Manage Users section then click on the Trash icon of the user you want to delete.

Finally in the modal that appears, confirm that you really want to delete the user.

Once you delete the user, you cannot retrieve any information from it, so be sure before performing this action.

Chat Conversation #back to top

Telegram bot builder gives you the ability to take over the conversation with your customers by sending messages through your bot.

To do this, from the Chat Conversation menu, select the bot for which you want to send particular messages, a new tab will open where you can have a conversation with your correspondent.
